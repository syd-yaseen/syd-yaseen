# Syed Yaseen | AI Solutions Architect & Founder

**Building Intelligence for Physical Reality.**

I build and deploy Computer Vision as a Service (CVaaS) systems that operate at the edge. My focus is entirely on architecting pipelines that bridge raw RTSP video streams, edge NPUs, and cloud backends to solve physical operational bottlenecks—from retail loss prevention to facility traffic flow.

### ⚙️ Current Focus
- **Edge AI Overlays:** Architecting hybrid CV pipelines that cascade lightweight edge triggers into heavy backend Pose Estimation and multi-object tracking.
- **Smart Operations (CVaaS):** Deploying real-time spatial correlation systems for suspicious loitering, staff-customer interaction, and queue management.
- **Hardware-to-Software Integration:** Breaking vendor locks by translating raw NPU data (Axera, NVIDIA Jetson) into scalable, asynchronous software dashboards.

### 🛠️ Tech Stack & Infrastructure

- **Backend & Data Pipeline:** Python, Django, PostgreSQL, FastAPI, Node-RED, Redis, WebSockets, REST APIs
- **Edge & Compute:** NVIDIA Jetson (Orin series), High-VRAM GPU Training Servers, RTSP Stream Processing
